Parses a string with a version spec in NuGet format. See http://docs.nuget.org/docs/reference/versioning Exception if the string does not contain a valid version spec in NuGet format. The version string is either a simple version or an arithmetic range e.g. 1.0 --> 1.0 ≤ x (,1.0] --> x ≤ 1.0 (,1.0) --> x < 1.0 [1.0] --> x == 1.0 (1.0,) --> 1.0 < x (1.0, 2.0) --> 1.0 < x < 2.0 [1.0, 2.0] --> 1.0 ≤ x ≤ 2.0
